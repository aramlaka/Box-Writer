# Box-Writer
Creates 3D "text boxes" using ASCII characters.

Currently implementing JavaFX, a lot more work to do!

Examples:

                 x  o  B     t  x  e  T
                o                    ee
               B                    x x
                                   t  t
             t                         
            x                    B    B
           e                    o     o
          T  e  x  t     B  o  x      x
          e                    o     o
          x                    B    B
          t        Label            
                               t  t
          B                    x x
          o                    ee
          x  o  B     t  x  e  T
